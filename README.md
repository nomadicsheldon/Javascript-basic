# Javascript-basic

## Variables
1. let - you can change the value defined by let.
2. const - you can't change the value defined by const.

## Primitive Types

### Numbers

### remainder / modulus operator
```javascript
10 % 6
```
output = 4

### exponential operator
```javascript
2 ** 3
```
output = 8

### NaN
```javascript
0/0 //NaN
1 + NaN //NaN
```
### Infinity
```javascript
1/0 //Infinity
-1/0 //-Infinity
```
